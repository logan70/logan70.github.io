# Zepto框架介绍

**为什么要使用Zepto**

- 无缝接入和改造现有的使用jQuery的项目
- 更好的代码执行效率

**对比其他框架的优势**

- 更低廉的学习成本
- 比较轻量化，没有提供复杂的模式

# Zepto框架核心API

## Zepto core

```js
$()
$(selector, [context])  ⇒ collection
$(<Zepto collection>)  ⇒ same collection
$(<DOM nodes>)  ⇒ collection
$(htmlString)  ⇒ collection
$(htmlString, attributes)  ⇒ collection v1.0+
Zepto(function($){ ... }) 
```
